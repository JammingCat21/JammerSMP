## Economy Survival with a additional special world where you can experience my plugins.
- Inventories carry between worlds.
- Things may be busted or unbalanced but fun is the main driver.
- This server logs IPs as every server does. You accept this by joining. This is used for security reasons.

## Gameplay
- Normal Difficulty.
- World Size is 30m.
- Server Reset is on Dec 31.
- You can sell everything but can only buy certain things.

## Server Restart Time

- 9am ~ London
- 10am ~ Berlin
- 6pm ~ Japan
- 8pm ~ Sydney
- 12am ~ Alaska
